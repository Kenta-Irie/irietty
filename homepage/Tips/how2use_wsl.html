WSLを使うにあたって

WSL（Windows Subsystem for Linux）を利用すると、Windows上でLinuxを動かすことができる。Windows10から実装された。

Ubuntuの初期設定

①ターミナル画面に移ると、インストールのため数分待つよう指示される。

②インストール終了後、ユーザー名とパスワードを設定する。これはsudoコマンド使用時のパスワードとなる。

パッケージのインストール

$ sudo apt install パッケージ名

GUIでEmacsやGrADSを使う準備

$ sudo apt install x11-apps x11-utils x11-xserver-utils dbus-x11
export DISPLAY=localhost:0.0

Emacsを使う

$ sudo apt install emacs

VS Codeを使う

GrADSを使う

$ sudo apt install grads

ディレクトリ構造

cd /mnt/c/Users/(your_accout_name)/Downloads/

トラブルシューティング

$ service ssh status
$ service ssh restart
